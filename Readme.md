## 更适合中国Misc手体质的套题生成器

之前看到某个地方要收600道题目，突发奇想写了个烂题生成器，目前还不完善，以后会多加点模块进来，现在只有6个

```
lsb
msb
生成二维码并按照像素转为01
多重base加密
将文件hex值直接写入图片rgb
弱口令压缩包
```

使用本项目：

```
pip install -r requirements.txt
python3 main.py
```

然后选择要套的层数以及flag值就会返回随机套娃之后的文件的base64值

pics文件夹用来存放各种隐写所需要的图片载体

dicts用来存放所需使用的字典文件

scripts用来存放模块

添加模块只需写个加密函数放入scripts文件夹然后改一下main.py
